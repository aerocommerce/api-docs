# View Category

## Attributes

| Name         | Type   | Description                                             |
|--------------|--------|---------------------------------------------------------|
| `id`         | int    | The id of the category                                  |
| `name`       | string | The name of the category, e.g. coats                    |
| `breadcrumb` | string | The breadcrumb path for the category, e.g. mens > coats |
